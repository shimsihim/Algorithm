#문제풀이
단순 BFS문제 <br/>
BFS에서 추가적인 사항은 전체를 순회하여 Q에 넣을 때 1과의 거리가 MAX와 같다면 +1을 해주고 MAX보다 크다면 CNT를 초기화 후 MAX를 갱신한다.<br/>

## 시간을 줄이기 위해 한 노력
int vis =1 변수를 정의하였다. 이는 내가 방문한 노드의 개수를 뜻하며 첫 노드는 방문한 것이므로 1로 시작.<br/>
반복을 돌며 q에 새로 삽입 시 이는 방문한 것과 다름없으므로 +1을 해준다. 이때 vis가 노드의 총 개수 n과 같아지면 while문을 빠져나온다.<br/>
이를 통해서 q안에 남아있는 노드들이 존재하여도 while문을 빠르게 빠져나오며 복잡한 간선일 경우 유효하다고 볼 수 있다.

<p>n개의 노드가 있는 그래프가 있습니다. 각 노드는 1부터 n까지 번호가 적혀있습니다. 1번 노드에서 가장 멀리 떨어진 노드의 갯수를 구하려고 합니다. 가장 멀리 떨어진 노드란 최단경로로 이동했을 때 간선의 개수가 가장 많은 노드들을 의미합니다.</p>

<p>노드의 개수 n, 간선에 대한 정보가 담긴 2차원 배열 vertex가 매개변수로 주어질 때, 1번 노드로부터 가장 멀리 떨어진 노드가 몇 개인지를 return 하도록 solution 함수를 작성해주세요.</p>
